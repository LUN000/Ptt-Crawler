# Ptt-Crawler
輸入選擇版名、爬取頁數\
抓取標題、時間、推數、連結、內文\
多種輸入關鍵字查詢標題
  1. 同時多關鍵字: or
  2. 滿足多關鍵字: and
  3. 用作者id查詢
  4. 一般單關鍵字

可透過關鍵字印出文章內容
